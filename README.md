# Conversor de Unidades

## Sobre o projeto
Este projeto é uma aplicação web simples que permite converter valores entre diferentes unidades de comprimento (metros, milímetros, centímetros e quilômetros).  

________________________________________
🎯 Objetivo  
Facilitar a conversão rápida e intuitiva de unidades de medida diretamente no navegador.  

________________________________________
✨ Funcionalidades  
- Conversão entre **m, mm, cm e km**  
- Exibição do resultado em tempo real no campo de saída  
- Mensagem explicativa mostrando a equivalência entre as unidades  
- Interface amigável e responsiva  

________________________________________
🛠️ Tecnologias utilizadas  
- **HTML5** para estruturação da página  
- **CSS3** para estilização e design responsivo  
- **JavaScript (ES6)** para lógica de conversão e interação com o usuário  

________________________________________
📂 Estrutura do projeto  
```
├── index.html        # Página principal
├── styles.css        # Estilos da aplicação
├── scripts.js        # Lógica de conversão
└── README.md         # Documentação do projeto
```
